FAIR-biomed

Biomedical data are distributed in open databases and knowledge bases. The FAIR-biomed browser extension allows you to access snippets of this data from any web page.

Select some text in your browser window. With the mouse, open a context menu and select the extension, right-click > FAIR-biomed search. Alternatively, press Ctr+Shift+Z on the keyboard. A popup window will show a collection of data resources that may have additional information pertaining to the query.

Additional information can be found on the FAIR-biomed website.

Plugin library

Settings

The order in which plugins appear in popups is determined by:

  • whether a plugin can respond to a query, and
  • the number of times a plugin was used in the past.

Information about past usage is stored in the browser local storage.

Reset plugin usage

By default, the extension popup displays a list of plugins and waits for a selection before fetching information from remote databases. Alternative, it can automatically fetch data from the last-used component.
