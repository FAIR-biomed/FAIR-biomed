FAIR-biomed

Biomedical data are distributed in open databases and knowledge bases. The FAIR-biomed browser extension allows you to access snippets of this data from any web page.

Select some text in your browser window. With the mouse, open a context menu and select the extension, right-click > FAIR-biomed search. Alternatively, press Ctr+Shift+Z on the keyboard. A popup window will show a collection of data resources that may have additional information pertaining to the query.

Plugin library

Plugin ranking

The order in which plugins appear in popups is determined by whether each plugin can respond to the query and by the number of times each plugin was used in the past.
